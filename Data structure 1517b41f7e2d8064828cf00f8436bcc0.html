Data Structures in Python

Python offers versatile data structures that are essential for solving real-world problems effectively. In this section, we’ll explore Lists, Tuples, Sets, and Dictionaries with examples and use cases.

1. Lists

A list is an ordered, mutable collection of items.

Key Features:

  • Allows duplicate items.
  • Elements can be of different data types.

Example:

# Creating a list
fruits = ["apple", "banana", "cherry"]

# Accessing elements
print(fruits[0])  # Output: apple

# Adding elements
fruits.append("orange")
print(fruits)  # Output: ['apple', 'banana', 'cherry', 'orange']

# Removing elements
fruits.remove("banana")
print(fruits)  # Output: ['apple', 'cherry', 'orange']

# Slicing
print(fruits[1:])  # Output: ['cherry', 'orange']

Use Case: Storing and managing a collection of items, such as a shopping list or student names.

2. Tuples

A tuple is an ordered, immutable collection of items.

Key Features:

  • Faster than lists because they are immutable.
  • Used when you need a fixed set of elements.

Example:

# Creating a tuple
coordinates = (10, 20, 30)

# Accessing elements
print(coordinates[1])  # Output: 20

# Tuples are immutable
# coordinates[1] = 25  # This will raise an error

Use Case: Representing fixed collections, like geographical coordinates or RGB color values.

3. Sets

A set is an unordered, mutable collection of unique elements.

Key Features:

  • Does not allow duplicate items.
  • Useful for membership tests and removing duplicates.

Example:

# Creating a set
numbers = {1, 2, 3, 4}

# Adding elements
numbers.add(5)
print(numbers)  # Output: {1, 2, 3, 4, 5}

# Removing duplicates
unique_numbers = set([1, 2, 2, 3, 4, 4])
print(unique_numbers)  # Output: {1, 2, 3, 4}

Use Case: Removing duplicates from a list or performing set operations like union and intersection.

4. Dictionaries

A dictionary is an unordered collection of key-value pairs.

Key Features:

  • Keys are unique and immutable.
  • Values can be of any data type.

Example:

# Creating a dictionary
student = {"name": "Alice", "age": 21, "grade": "A"}

# Accessing values
print(student["name"])  # Output: Alice

# Adding a key-value pair
student["major"] = "Computer Science"
print(student)  # Output: {'name': 'Alice', 'age': 21, 'grade': 'A', 'major': 'Computer Science'}

# Iterating over a dictionary
for key, value in student.items():
    print(f"{key}: {value}")

Use Case: Storing and managing data that requires a label, such as a contact list or configuration settings.

List Comprehension

Python allows concise ways to create and manipulate lists.

Example:

# Creating a list of squares
squares = [x**2 for x in range(5)]
print(squares)  # Output: [0, 1, 4, 9, 16]
