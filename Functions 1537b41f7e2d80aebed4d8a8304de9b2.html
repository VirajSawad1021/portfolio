Functions in Python

Functions are one of the core building blocks in Python. They allow you to group code into reusable blocks, making your program more modular, organized, and easier to maintain. Functions can take inputs, process them, and return outputs. By defining functions, you can reduce repetition and improve the clarity of your code.

1. Defining and Calling Functions

In Python, functions are defined using the def keyword followed by the function name and parentheses.

Example:

# Defining a function
def greet(name):
    print(f"Hello, {name}!")

# Calling the function
greet("Viraj")  # Output: Hello, Viraj!

Key Points:

  • def is used to define a function.
  • Parameters are passed inside parentheses and are used in the function body.
  • The function is called by its name followed by parentheses.

2. Return Values

Functions can return values using the return keyword. This allows you to pass data back to the caller.

Example:

# Function that returns a value
def add_numbers(a, b):
    return a + b

# Calling the function and using the returned value
result = add_numbers(3, 5)
print(result)  # Output: 8

Key Points:

  • The return statement ends the function and sends the result back to the caller.
  • A function can return multiple values by separating them with commas.

3. Default Parameters

Functions can have default values for parameters. If no argument is passed for a parameter with a default value, the default value is used.

Example:

# Function with default parameters
def greet(name="Guest"):
    print(f"Hello, {name}!")

greet("Viraj")  # Output: Hello, Viraj!
greet()         # Output: Hello, Guest!

Key Points:

  • Default parameters are defined by assigning values to them in the function definition.
  • Parameters with default values should come after parameters without default values.

4. Variable Number of Arguments

Sometimes, you don't know how many arguments will be passed to the function. In such cases, you can use the *args and **kwargs syntax.

  • args allows a function to accept any number of positional arguments.
  • *kwargs allows a function to accept any number of keyword arguments.

Example with *args:

# Function with *args
def sum_numbers(*args):
    return sum(args)

result = sum_numbers(1, 2, 3, 4)
print(result)  # Output: 10

Example with **kwargs:

# Function with **kwargs
def display_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

display_info(name="Viraj", age=21, city="Kathmandu")
# Output:
# name: Viraj
# age: 21
# city: Kathmandu

Key Points:

  • args collects extra positional arguments into a tuple.
  • *kwargs collects extra keyword arguments into a dictionary.

5. Lambda Functions

A lambda function is a small, anonymous function defined with the lambda keyword. It is typically used for short, simple functions.

Example:

# Lambda function to add two numbers
add = lambda x, y: x + y

print(add(3, 5))  # Output: 8

Key Points:

  • Lambda functions are defined with lambda followed by parameters and an expression.
  • They are often used for one-liner functions.

6. Recursive Functions

A recursive function is one that calls itself. Recursive functions are used to solve problems that can be broken down into smaller subproblems.

Example:

# Recursive function to calculate factorial
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120

Key Points:

  • Recursion is useful for problems like factorial, Fibonacci sequence, etc.
  • Ensure that there is a base case to terminate the recursion and avoid infinite recursion.

Why Use Functions?

  • Modularity: Functions allow you to break down complex problems into smaller, manageable parts.
  • Reusability: Functions can be reused across different parts of a program or in multiple projects.
  • Readability: Code is easier to understand when it is organized into well-named functions that describe their purpose.
